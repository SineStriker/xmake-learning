# XMake Learning

XMake 构建系统学习笔记

## 简介

XMake 整体生态还不算壮大，官方文档体验较差，搜索引擎与 ChatGPT 不能很好地发挥解答作用，遂求教各方大佬，整合各种文献，将各种经验总结于此。

由于 CMake 与 XMake 具有极大差异，因此习惯 CMake 以后学习 xmake 会有非常强烈的不适应感，本文也总结了一些在 XMake 中实现 CMake 的功能的经验。
